# eKart E-commerce Backend API

A backend API project for the eKart E-commerce website using Java and Spring Framework

# Tech Stack :
<ul>
    <li>Spring Boot</li>
    <li>Spring Web</li>
    <li>Spring Data JPA</li>
    <li>Spring Data Rest</li>
    <li>Swagger</li>
    <li>H2 and Postgres DB</li>
</ul>
